# Sample Exam - Color Preferences

This is an example exam file showing all supported question formats.

---

Q001. What is the favorite color of me?

A. Blue
B. Red
C. Green
D. Orange

Answer: D

---

Q002. What is the capital of France?

Answer: Paris

---

Q003. What are the primary colors?

A. Blue
B. Red
C. Green
D. Yellow

Answer: A, B, D

---

Q004. Which programming languages are commonly used for web development?

A. Python
B. JavaScript
C. TypeScript
D. Rust
E. HTML

Answer: B, C, E

---

Q005. What is 2 + 2?

A. 3
B. 4
C. 5
D. 6

Answer: B

---

Q006. The Earth revolves around which celestial body?

A. Moon
B. Mars
C. Sun
D. Jupiter

Answer: C

---

Q007. What is the chemical symbol for water?

Answer: H2O

---

Q008. Which of the following are valid JavaScript variable declarations?

A. var x = 1
B. let y = 2
C. const z = 3
D. int w = 4

Answer: A, B, C

---

Q009. What year did World War II end?

A. 1943
B. 1944
C. 1945
D. 1946

Answer: C

---

Q010. Name the largest planet in our solar system.

Answer: Jupiter
